# RISC-V-CPU in SystemVerilog
This project is a from scratch implementation of a RISC-V RV32I CPU using SystemVerilog

## Goals
-Learn CPU Architecture deeply
-Build a working RV32I singe-cycle CPU
-Extend to a pipelined CPU
-Run programs on an FPGA
